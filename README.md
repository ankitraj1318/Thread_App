# Thread App

## Tech Stack Used
    
-   🌟 Tech stack: MERN + Socket.io + Chakra UI

## Feature List:
-   🎃 Authentication & Authorization with JWT
-   📝 Create Post
-   🗑️ Delete Post
-   ❤️ Like/Unlike Post
-   💬 Comment to a Post
-   👥 Follow/Unfollow Users
-   ❄️ Freeze Your Account
-   🌓 Dark/Light Mode
-   📱 Completely Responsive
-   💬 Chat App With Image Support
-   👀 Seen/Unseen Status for Messages
-   🔊 Notification sounds
-   ⭐ Deployment for FREE
